What it captures

Reasoning, extracted from everywhere it hides

Archeon reads the trail your team already leaves behind and turns it into a queryable decision graph.

Git commit history

Hashes, authors, dates, messages, changed files, and diff summaries — the raw record of every change, tagged as cited evidence.

Pull requests & issues

PR bodies, review comments, and the issues they link to — pulled straight from the GitHub API where most rationale actually lives.

ADRs, docs & READMEs

Architecture decision records and doc write-ups split by heading, so long-form reasoning becomes individually retrievable.

Inline code rationale

Comments and docstrings that explain why a line exists — captured as inferred signal alongside the harder evidence.

Cited, always

Every chunk is stamped with a [source= locator=] header so provenance survives all the way into the answer.

Incremental & edge-safe

Re-run on only new commits. Merge commits, empty commits, binary files, and repos without PRs are all handled gracefully.

How it works

From raw history to a cited answer

Two halves: ingestion turns your repo into a decision graph, and the query engine answers questions against it.

Ingest

Extractors emit {source, content, metadata} records from git, PRs, docs, and comments — normalized into newline-delimited JSON.

Build the graph

Cognee's cognify() extracts a typed graph — Decision, Context, Consequence, and Evidence nodes joined by motivated-by and results-in edges.

Recall

archeon why runs two-pass retrieval: one pass synthesizes the answer, the other recovers the source chunks behind it.

Trust, made explicit

Every answer is confidence-scored

Archeon never bluffs. Each response is tagged so you know exactly how much to trust it — cited > inferred > unknown.

[cited]

Backed by evidence

The answer is grounded in a recovered commit, PR, ADR, or issue. The strongest tier — you can click through to the source.

[inferred]

Synthesized from the graph

A reasonable answer assembled from the decision graph, but with no single attributable source to cite.

[unknown]

An honest gap

No memory was found. Archeon reports the gap instead of hallucinating an answer — a missing decision you can go document.

Quick start

Up and running in four commands

Requires Python 3.10+ and git. Point it at a repo and start asking why.

1
Install

Clone and install with the Cognee extras.

2
Check status

Confirm the CLI and Cognee surface are live.

3
Ingest a repo

Extract history into the decision graph.

4
Ask why

Query in natural language and get a cited answer.

terminal 
# 1 · install
git clone https://github.com/kishiagaytano/archeon.git
cd archeon && pip install -e ".[cognee,lifecycle,dev]"

# 2 · verify
archeon status

# 3 · ingest history → decision graph
archeon ingest demo/atlas-api

# 4 · ask why
archeon why "why did we choose FastAPI over Flask?"
